## 사전작업

Android 빌드시,
android emulator 혹은
Android Studio의 AVD Manager에 가상머신 추가

ios 빌드시,
xcode 설치 및 cocoapod 설치

## Available Scripts

프로젝트 경로상에서, 아래와 같은 스크립트 실행이 가능합니다:

### `npm run ios`

아이폰 디바이스로 빌드합니다.

### `npm run android`

안드로이드 디바이스로 빌드합니다.
